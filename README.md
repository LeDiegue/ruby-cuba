# ruby-cuba
Learn Ruby On Cuba

#### Dependencies
##### Cuba
```
gem install cuba
```
#### Run Run
First, you need install dependencies. Please, run the following command:
```
dep install
```
and to finish, run:
```
rackup
```

Ready! :D.

You can check in localhost on port 9292 by default. [localhost:9292](http://localhost:9292/)
